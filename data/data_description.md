| Biến | Định nghĩa | Công thức |
|---|---|---|
| DSRI | Chỉ số khoản phải thu so với doanh thu | (Khoản phải thu / Doanh thu)_t ÷ (Khoản phải thu / Doanh thu)_{t−1} |
| GMI | Chỉ số biên lợi nhuận gộp | [(Doanh thu − Giá vốn)/Doanh thu]_{t−1} ÷ [(Doanh thu − Giá vốn)/Doanh thu]_t |
| AQI | Chỉ số chất lượng tài sản | [1 − (Tài sản ngắn hạn + TSCĐ)/Tổng tài sản]_t ÷ [1 − (Tài sản ngắn hạn + TSCĐ)/Tổng tài sản]_{t−1} |
| SGI | Chỉ số tăng trưởng doanh thu | Doanh thu_t ÷ Doanh thu_{t−1} |
| DEPI | Chỉ số khấu hao | (Khấu hao_{t−1}/(TSCĐ_{t−1}+Khấu hao_{t−1})) ÷ (Khấu hao_t/(TSCĐ_t+Khấu hao_t)) |
| SGAI | Chỉ số chi phí bán hàng và quản lý | (Chi phí bán hàng & QLDN / Doanh thu)_t ÷ (Chi phí bán hàng & QLDN / Doanh thu)_{t−1} |
| LVGI | Chỉ số đòn bẩy tài chính | (Nợ / Tổng tài sản)_t ÷ (Nợ / Tổng tài sản)_{t−1} |
| TATA | Tổng accruals trên tổng tài sản | (Lợi nhuận sau thuế − Dòng tiền HĐKD) / Tổng tài sản |
| RSST_Accruals | Chất lượng accrual | (ΔVốn lưu động + ΔTài sản hoạt động dài hạn + ΔTài sản tài chính) / Tổng tài sản bình quân |
| Delta_Receivables | Thay đổi khoản phải thu | (Khoản phải thu_t − Khoản phải thu_{t−1}) / Tổng tài sản |
| Delta_Inventory | Thay đổi hàng tồn kho | (Tồn kho_t − Tồn kho_{t−1}) / Tổng tài sản |
| Delta_ROA | Thay đổi tỷ suất sinh lời trên tài sản | ROA_t − ROA_{t−1} |
| Soft_Assets | Tỷ lệ tài sản kém thanh khoản | (Tổng tài sản − TSCĐ − Tiền) / Tổng tài sản |
| ROA | Tỷ suất sinh lời trên tổng tài sản | Lợi nhuận sau thuế / Tổng tài sản |
| ROE | Tỷ suất sinh lời trên vốn chủ sở hữu | Lợi nhuận sau thuế / Vốn chủ sở hữu |
| Net_Profit_Margin | Biên lợi nhuận ròng | Lợi nhuận sau thuế / Doanh thu |
| Gross_Profit_Margin | Biên lợi nhuận gộp | (Doanh thu − Giá vốn) / Doanh thu |
| Revenue_Growth | Tăng trưởng doanh thu | (Doanh thu_t − Doanh thu_{t−1}) / Doanh thu_{t−1} |
| Debt_to_Assets | Tỷ lệ nợ trên tổng tài sản | Tổng nợ / Tổng tài sản |
| Debt_to_Equity | Tỷ lệ nợ trên vốn chủ sở hữu | Tổng nợ / Vốn chủ sở hữu |
| Receivables_to_Revenue | Tỷ lệ khoản phải thu trên doanh thu | Khoản phải thu / Doanh thu |
| Receivables_to_Assets | Tỷ lệ khoản phải thu trên tổng tài sản | Khoản phải thu / Tổng tài sản |
| Inventory_to_Assets | Tỷ lệ hàng tồn kho trên tổng tài sản | Hàng tồn kho / Tổng tài sản |
| Current_Assets_to_Total_Assets | Tỷ lệ tài sản ngắn hạn trên tổng tài sản | Tài sản ngắn hạn / Tổng tài sản |
| CFO_to_Assets | Dòng tiền HĐKD trên tổng tài sản | Dòng tiền từ HĐKD / Tổng tài sản |
| CFO_to_Revenue | Dòng tiền HĐKD trên doanh thu | Dòng tiền từ HĐKD / Doanh thu |
| Accruals_to_Assets | Tổng accruals trên tổng tài sản | (Lợi nhuận sau thuế − Dòng tiền HĐKD) / Tổng tài sản |
| Working_Capital_to_Assets | Vốn lưu động trên tổng tài sản | (Tài sản ngắn hạn − Nợ ngắn hạn) / Tổng tài sản |
| Firm_Size | Quy mô doanh nghiệp | ln(Tổng tài sản) |
| Issue | Phát hành cổ phiếu | 1 nếu doanh nghiệp phát hành cổ phiếu trong năm, ngược lại = 0 |
| Company_Age | Tuổi doanh nghiệp | Năm hiện tại − Năm thành lập |
| Auditor_Change | Thay đổi kiểm toán viên | 1 nếu thay đổi kiểm toán viên so với năm trước, ngược lại = 0 |
| Board_Independence | Tỷ lệ thành viên HĐQT độc lập | Số thành viên độc lập / Tổng số thành viên HĐQT |
